CSS3 Background

CSS3 contains a few new background properties, which allow greater control of the background element.In this chapter you will learn how to add multiple background images to one element. You will also learn about the following new CSS3 properties:

  • background-size: w, h / contain / cover
  • background-origin: border-box/padding-box/content-box
  • background-clip
